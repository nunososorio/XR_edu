# GitHub Pages - Quick Start

**Your website will be at:** https://nunososorio.github.io/XR_EDU

## Three Easy Steps

### 1️⃣ Push to GitHub (2 minutes)

```powershell
cd c:\Users\nunoo\Documents\GitHub\XR_edu
git add .
git commit -m "Add GitHub Pages deployment"
git push origin main
```

### 2️⃣ Enable GitHub Pages (1 minute)

1. Go to: https://github.com/nunososorio/XR_EDU/settings/pages
2. Under **Source:** Select **GitHub Actions**
3. Click **Save** (if button appears)

### 3️⃣ Wait & Visit (3 minutes)

1. Check: https://github.com/nunososorio/XR_EDU/actions
2. Wait for green ✓ next to "Deploy Documentation to GitHub Pages"
3. Visit: **https://nunososorio.github.io/XR_EDU** ✨
