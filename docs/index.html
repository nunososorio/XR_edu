What is This?

A collaborative bookmark management system built on Linkwarden (GNU AGPL v3.0), specifically designed for collecting, organizing, and sharing XR applications, educational videos, games, and learning experiences with pedagogical value.

Self-hosted, open-source, with complete data ownership. No subscriptions, no analytics tracking, no third-party control. Just your own powerful knowledge management platform.

Key Features

  • Bookmark collection with automatic preservation
  • Full-text search powered by MeiliSearch
  • Team collaboration with permission management
  • XR platform metadata (Quest, Vision Pro, WebXR, etc.)
  • Pedagogical information tracking
  • Age/grade level targeting
  • Accessibility feature documentation
  • Reader view with annotations and highlighting
  • Public collections and sharing
  • REST API for integrations
  • Docker deployment ready
  • PostgreSQL + MeiliSearch stack

Quick Start

  • Clone the GitHub repository
  • Run docker-compose up -d in the linkwarden directory
  • Visit http://localhost:3000
  • Create your first collection
  • Start organizing XR educational content
